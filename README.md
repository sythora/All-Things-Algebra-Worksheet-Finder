# Project **ATA**
 (All Things Algebra) Worksheet finder is an NodeJS web app that finds worksheets from ATA. The worksheets are **NOT** the previews and the full worksheet with the answers.

## 🧠 Why I made this  
You used to be able to JUST remove preview from the url and it works, doesn't no more + I don't feel like doing 15 questions of a long formula

## 📅 When I made this  
April 17 at 8:35 AM (I started working on it), I'm making this on a crappy chromebook and github spaces.

## 🔍 How do you use it?

1. get worksheet name  
2. Find the page on google
4. open the page  
5. grab the link (Ex: https://allthingsalgebra.com/product/quadratic-functions-standard-form-vertex-form-and-graphs-triples-activity/)
6.  Remove  the link and everything else so you're left with the first 4-5 words: (Ex: quadratic-functions-standard-form-vertex), Yes it's sloppy but I'll make it better one day
7. input the URL into the web app
8. Click on the link that doesn't have COVER or PREVIEW and ends in PDF.
